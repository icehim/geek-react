# react实战项目

> **前言:**
>
> 此项目是用React开发的PC端项目
>
> 技术栈:React + React hooks + React-router + Redux + Redux-thunk + Ant Design + axios +Scss
>
> 账号：13911111111 密码：246810

>**安装环境：**
>
>需要在node环境下，在package.json所在的文件夹中npm i下载依赖，下载完成后npm run start 启动项目在`http://localhost:3000/login`即可访问
